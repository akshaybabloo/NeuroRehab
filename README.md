# NeuroRehab
A VR game for Neurorehabilitation.

> Note: This application is under development.

# Controls

1. Move ball left - <kbd>E</kbd>
2. Move ball right - <kbd>R</kbd>
3. Look around - Mouse or Oculus
4. Character movement - Not applicable
